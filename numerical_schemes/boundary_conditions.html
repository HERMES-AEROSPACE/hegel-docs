Boundary Conditions

HEGEL supports a comprehensive set of boundary conditions for inviscid (Euler) and viscous (Navier-Stokes) flows. Boundary conditions are specified in a separate BC file referenced from the main input file.

Inviscid (Euler) Boundary Conditions

Symmetry / Slip Wall

Reflects the normal velocity component while preserving tangential components and thermodynamic state. Used for symmetry planes and inviscid walls.

BC_TYPE = SYMMETRY

No parameters required.

Farfield

Characteristic-based treatment for farfield boundaries. Incoming characteristics are set from freestream conditions; outgoing characteristics are extrapolated from the interior.

BC_TYPE = FARFIELD

Parameters: Freestream density, velocity, pressure.

Source: src/fluid_eu_ns/EU_bc_face_farfield.F90

Supersonic Inlet / Outlet

At a supersonic inlet, all characteristics enter the domain — all flow variables are specified from boundary data:

BC_TYPE = SUP_IN

At a supersonic outlet, all characteristics leave the domain — all variables are extrapolated (zero-gradient):

BC_TYPE = SUP_OUT

Subsonic Inlet / Outlet

Subsonic Inlet (Static Conditions)

Reference

    1. Whitfield, J. M. Janus, AIAA Paper 84-1552 (1984).

Imposes static pressure and temperature. Density extrapolated from interior.

BC_TYPE = SUB_IN
P       = 101325.0
T       = 300.0

Source: src/fluid_eu_ns/EU_bc_face_in.F90

Subsonic Inlet (Total Conditions)

Reference

    1. Carlson, NASA/TM-2011-217181 (2011).

Imposes total (stagnation) pressure and temperature. Flow direction derived from interior characteristics.

BC_TYPE = SUB_IN_TOT
P0      = 200000.0
T0      = 500.0

Source: src/fluid_eu_ns/EU_bc_face_in_tot.F90

Subsonic Inlet (Mass Flow Rate)

Imposes mass flow rate and temperature. Useful for combustor and ICP torch inlets.

BC_TYPE  = SUB_IN_MDOT
MDOT     = 0.5
T        = 300.0

For multi-species (NLTE): additional RHOI or XI parameters specify species composition.

Source: src/fluid_eu_ns/EU_bc_face_in_mdot.F90

Subsonic Outlet

Imposes static back-pressure; all other quantities extrapolated from interior.

BC_TYPE = SUB_OUT
P       = 101325.0

Source: src/fluid_eu_ns/EU_bc_face_out.F90

Ambivalent Inlet/Outlet

Dynamically switches between inlet and outlet behaviour based on local flow direction. Useful when the flow direction at the boundary is not known a priori (e.g., separated regions).

BC_TYPE = SUB_IN_OUT
P0      = 200000.0
T0      = 500.0
P       = 101325.0

Source: src/fluid_eu_ns/EU_bc_face_in_out.F90

Viscous (Navier-Stokes) Wall Conditions

Isothermal No-Slip Wall

Velocity set to zero; wall temperature specified; pressure extrapolated.

BC_TYPE = NO_SLIP_WALL_ISO
T       = 300.0

Source: src/fluid_eu_ns/NS_bc_face_wall_iso.F90

Adiabatic No-Slip Wall

Velocity set to zero; zero heat flux at wall. Temperature recovers to the adiabatic wall temperature.

BC_TYPE = NO_SLIP_WALL_AD

No additional parameters.

Isothermal Wall with Gas-Surface Interactions (NLTE)

No-slip wall with catalytic surface reactions. Species composition at the wall is determined by surface chemistry. Requires NLTE fluid model.

BC_TYPE = NO_SLIP_WALL_ISO_GSI
T       = 1500.0

Source: src/fluid_eu_ns_nlte/NS_NLTE_bc_face_wall_iso_gsi.F90

Mixed Isothermal/Adiabatic Wall (NLTE)

Translational temperature set to wall value (isothermal); vibrational and electronic temperatures set to adiabatic recovery. Decouples temperature modes at the wall.

BC_TYPE = NO_SLIP_WALL_AD_TINT
T       = 1000.0

Source: src/fluid_eu_ns_nlte/NS_NLTE_bc_face_wall_iso_ad.F90

Moving Adiabatic Wall

No-slip wall with prescribed wall velocity.

BC_TYPE = MOV_NO_SLIP_WALL_AD
VEL     = 100.0  0.0  0.0

Source: src/fluid_eu_ns/NS_bc_face_wall_mov.F90

Blowing Wall (Mass Injection)

Wall with prescribed normal velocity (transpiration). Used for ablation modelling and film cooling.

BC_TYPE = BLOW_WALL
VEL     = 0.0  0.1  0.0
T       = 2000.0

For NLTE: add species composition with RHOI or XI.

Source: src/fluid_eu_ns/NS_bc_face_wall_blow.F90

Radiative Equilibrium Wall

Wall temperature determined by the balance between conductive heat flux to the wall and radiative emission:

\[q_\text{cond} = \epsilon \sigma (T_w^4 - T_\text{ext}^4)\]
BC_TYPE  = NO_SLIP_RAD_EQ_WALL
EMIS     = 0.85
TEMP_EXT = 300.0

Source: src/fluid_eu_ns/NS_bc_face_wall_rad_eq.F90

Radiative Equilibrium Wall with GSI (NLTE)

Combines radiative equilibrium temperature with catalytic surface chemistry.

BC_TYPE  = NO_SLIP_RAD_EQ_WALL_GSI
EMIS     = 0.85

Source: src/fluid_eu_ns_nlte/NS_NLTE_bc_face_wall_rad_eq_gsi.F90

LODI Non-Reflecting Boundary Conditions

Reference

  1. Motheau, Y. Mery, F. Nicoud, T. Poinsot, AIAA J. 55 (10), 2017.

    1. Thompson, J. Comput. Phys. 68, 1–24 (1987).

The LODI (Locally One-Dimensional Inviscid) approach uses characteristic decomposition to construct non-reflecting boundary conditions. Ghost-cell values are set such that incoming wave amplitudes are relaxed toward target conditions while outgoing waves pass freely.

LODI boundary conditions are essential for subsonic problems where spurious wave reflections from boundaries contaminate the interior solution (e.g., jet simulations, acoustic problems, unsteady flows).

Internal Boundaries

Block Interface

Internal boundary between computational blocks in a multi-block grid. Handled automatically by the grid module — not set by the user.

BC_TYPE = BLOCK_INTERFACE   # ID = -200

Ghost cell data is exchanged via MPI for inter-processor block interfaces.
