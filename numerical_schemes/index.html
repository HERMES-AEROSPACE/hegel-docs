Numerical Schemes

HEGEL employs a cell-centered finite volume method on multi-block structured grids. This section provides a comprehensive reference for every numerical scheme, flux function, time integrator, reconstruction method, boundary condition, and source term available in the solver.

Note

All scheme selection is performed at runtime through the input file. There are no compile-time flags that gate individual schemes (only entire physics modules such as NLTE or ICP are conditional). The input keyword for each option is listed in the tables below.

Quick Reference

The following table maps input keywords to the scheme they select. Use cross-references to jump to the detailed description.
