Spatial Discretization

HEGEL uses a cell-centered finite volume method on multi-block structured grids. High-order accuracy is achieved through reconstruction of interface states from cell-averaged values. This section documents all available reconstruction procedures, slope limiters, and reconstruction variable sets.

Piecewise Constant (CONST)

No reconstruction; left and right interface states are the cell averages themselves. First-order accurate. Robust but highly diffusive.

Input:

NUM_REC_PROC = CONST

Source files: src/fluid_eu_ns/EU_rec.F90 (no_rec)

When to use: Low-Mach ICP flows, startup phase of complex chemistry simulations, debugging.

MUSCL

Reference

  1. van Leer, J. Comput. Phys. 32, 101–136 (1979).

Monotone Upstream-centered Schemes for Conservation Laws. Achieves 2nd-order accuracy using a 4-point stencil \((i{-}1, i, i{+}1, i{+}2)\) with a slope limiter to suppress oscillations near discontinuities.

The implementation uses a two-argument form of the Van Albada limiter applied to forward and backward differences directly. The reconstructed left and right interface states are:

\[\begin{split}q_{i+1/2}^L &= q_i + \tfrac{1}{2}\,\psi(\Delta_+,\, \Delta_-,\, \epsilon) \\ q_{i+1/2}^R &= q_{i+1} - \tfrac{1}{2}\,\psi(\Delta_{++},\, \Delta_+,\, \epsilon)\end{split}\]

where \(\Delta_+ = q_{i+1} - q_i\), \(\Delta_- = q_i - q_{i-1}\), \(\Delta_{++} = q_{i+2} - q_{i+1}\), and the two-argument Van Albada function is:

\[\psi(a, b, \epsilon) = \frac{(b^2 + \epsilon)\,a + (a^2 + \epsilon)\,b} {a^2 + b^2 + 2\epsilon}\]

Here \(\epsilon\) is a small smoothing parameter to prevent division by zero.

Default limiter: Van Albada two-argument form (see Slope Limiters below).

Input:

NUM_REC_PROC = MUSCL
NUM_REC_VAR  = RHOVP    # or CHAR_RHOVP for characteristic MUSCL

Source files by model:

  • EU/NS: src/fluid_eu_ns/EU_rec_muscl_rhovp.F90 (primitive), EU_rec_muscl_char_rhovp.F90 (characteristic)

  • LTE: src/fluid_eu_ns_lte/EU_LTE_rec_muscl_rhovp.F90, EU_LTE_rec_muscl_rhovt.F90 (temperature-based)

  • NLTE: src/fluid_eu_ns_nlte/EU_NLTE_rec_muscl.F90

  • Core: src/utils/utils_rec.F90 (muscl_rec, muscl_no_lim_rec)

Positivity-preserving variants:

  • muscl_rec_pos_pres — enforces positivity of reconstructed density and pressure

  • muscl_rec_pos_pres_smooth(beta) — smooth positivity enforcement with parameter \(\beta\)

WENO3

3rd-order Weighted Essentially Non-Oscillatory reconstruction using two candidate stencils, each producing a 2nd-order interpolant. The final reconstruction is a convex combination weighted by smoothness indicators:

\[q_{i+1/2} = \sum_{k=0}^{1} \omega_k \, q^{(k)}_{i+1/2}\]

Optimal weights: \(d_0 = 1/3\), \(d_1 = 2/3\).

Smoothness parameter: \(\epsilon_\text{WENO} = 10^{-6}\).

Input:

NUM_REC_PROC = WENO3

Source files:

  • Core: src/utils/utils_rec.F90 (weno3_rec, weno3_optim_rec)

  • EU/NS: src/fluid_eu_ns/EU_rec_weno_rhovp.F90

WENO5

Reference

G.-S. Jiang, C.-W. Shu, J. Comput. Phys. 126, 202–228 (1996).

5th-order WENO reconstruction using three candidate stencils over a 6-point stencil \((i{-}2, \ldots, i{+}3)\):

\[q_{i+1/2} = \sum_{k=0}^{2} \omega_k \, q^{(k)}_{i+1/2}\]

Optimal weights: \(d_0 = 0.1\), \(d_1 = 0.6\), \(d_2 = 0.3\).

Non-linear weights:

\[\omega_k = \frac{\alpha_k}{\sum_j \alpha_j}, \qquad \alpha_k = \frac{d_k}{(\epsilon + \beta_k)^2}\]

where \(\beta_k\) are the smoothness indicators computed from undivided differences.

Input:

NUM_REC_PROC = WENO5
NUM_REC_VAR  = CHAR_RHOVP   # characteristic recommended for shocks

Source files:

  • Core: src/utils/utils_rec.F90 (weno5_rec, weno5_optim_rec)

  • EU/NS: src/fluid_eu_ns/EU_rec_weno_rhovp.F90, EU_rec_weno_char_rhovp.F90

  • LTE: src/fluid_eu_ns_lte/EU_LTE_rec_weno.F90

  • NLTE: src/fluid_eu_ns_nlte/EU_NLTE_rec_weno.F90

Non-uniform grid variant (WENO5_NU):

Uses divided differences for the smoothness indicators, automatically handling non-uniform cell spacing.

NUM_REC_PROC = WENO5_NU

Source: src/utils/utils_rec.F90 (weno5_nu_rec)

MP3

3rd-order Monotonicity-Preserving reconstruction. Uses the same stencil as WENO3 but applies an a posteriori monotonicity constraint via the median operation:

\[q_{i+1/2}^\text{MP} = \text{median}\bigl(q_i,\; q_{i+1/2}^\text{orig},\; q_\text{MP}\bigr)\]

Parameter: \(\alpha_\text{MP} = 2.0\) (controls curvature limiter).

Input:

NUM_REC_PROC = MP3

Source files:

  • Core: src/utils/utils_rec.F90 (mp3_rec)

  • EU/NS: src/fluid_eu_ns/EU_rec_mp_rhovp.F90, EU_rec_mp_char_rhovp.F90

MP5

Reference

  1. Suresh, H. T. Huynh, J. Comput. Phys. 136, 83–99 (1997).

5th-order Monotonicity-Preserving reconstruction. Achieves 5th-order accuracy in smooth regions while maintaining the monotonicity property near discontinuities. The unlimited 5th-order interpolant is:

\[q_{i+1/2}^\text{orig} = \frac{1}{60}\bigl(2\,q_{i-2} - 13\,q_{i-1} + 47\,q_i + 27\,q_{i+1} - 3\,q_{i+2}\bigr)\]

This is then constrained by the MP limiter using the median operation.

Parameter: \(\alpha_\text{MP} = 2.0\).

Input:

NUM_REC_PROC = MP5
NUM_REC_VAR  = RHOVP

Source files:

  • Core: src/utils/utils_rec.F90 (mp5_rec)

  • EU/NS: src/fluid_eu_ns/EU_rec_mp_rhovp.F90, EU_rec_mp_char_rhovp.F90

  • LTE: src/fluid_eu_ns_lte/EU_LTE_rec_mp.F90

  • NLTE: src/fluid_eu_ns_nlte/EU_NLTE_rec_mp.F90

Reconstruction Variable Sets

The choice of variables in which reconstruction is performed significantly affects the quality of the solution, especially near discontinuities.

Keyword

Variables

Notes

RHOVP

\((\rho, u, v, [w], p)\)

Default. Primitive variables.

CHAR_RHOVP

Characteristic projection

Projects to characteristic fields before limiting. Best for strong shocks.

RHOVT

\((\rho, u, v, [w], T)\)

LTE only. Temperature-based reconstruction.

Characteristic reconstruction (CHAR_RHOVP) projects the solution increments onto the eigenvectors of the flux Jacobian before applying the limiter, then projects back. This prevents spurious oscillations that arise when limiting different wave families (acoustic, entropy, shear) independently. Recommended when strong shocks or contact discontinuities are present.

Source files for eigensystem:

  • src/fluid_eu_ns/EU_eigsys.F90 (eigenvalues + left/right eigenvectors)

  • src/fluid_eu_ns/EU_transf_mat.F90 (transformation matrices)

NLTE variable sets:

For multi-species NLTE models, additional variable sets are available:

  • \((\rho_i, \mathbf{v}, T, T_k)\) — species partial densities with temperatures

  • \((\rho_i, \mathbf{v}, p, T_k)\) — species densities with pressure and internal temperatures

Slope Limiters

Limiters are applied within the MUSCL reconstruction to prevent oscillations. All limiters are implemented as symmetric functions \(\phi(r)\) where \(r\) is the ratio of consecutive gradients.

Source file: src/utils/utils_limiter.F90

Limiter

Formula

Properties

Van Albada

\(\phi(r) = \frac{r^2 + r}{r^2 + 1}\)

Smooth; default in HEGEL

Van Leer

\(\phi(r) = \frac{r + |r|}{1 + |r|}\)

Smooth; slightly more compressive

Minmod

\(\phi(r) = \max(0, \min(1, r))\)

Most diffusive TVD limiter

Superbee

\(\phi(r) = \max(0, \min(2r,1), \min(r,2))\)

Most compressive TVD limiter

Koren

\(\phi(r) = \max(0, \min(2r, (1+2r)/3, 2))\)

3rd-order on uniform grids

MC

\(\phi(r) = \max(0, \min(2r, (1+r)/2, 2))\)

Monotonized central

CHARM

\(\phi(r) = r(3r+1)/(r+1)^2\)

Non-TVD; smooth

HCUS

\(\phi(r) = 1.5(r+|r|)/(r+2)\)

Non-TVD

HQUICK

\(\phi(r) = 2(r+|r|)/(r+3)\)

Non-TVD

OSPRE

\(\phi(r) = 1.5(r^2+r)/(r^2+r+1)\)

Non-TVD; smooth

Note

The limiter is currently not independently selectable via the input file — MUSCL reconstruction uses Van Albada by default. The other limiters are available in the utility module for custom implementations.

Helper functions (src/utils/utils_rec.F90):

  • van_Albada_two_arg(a, b, eps) — two-argument form for direct use

  • minmod_two_arg(a, b) — two-argument minmod

  • median(x, y, z) — three-value median (used by MP schemes)
